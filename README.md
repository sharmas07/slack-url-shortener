
## Slack Bot URL Shortener

Shorten URLs directly from Slack channels with this bot! Send a message with "!short" followed by the URL, and get a shortened link. Built with Node.js, Slack's RTM API, and Web API. Explore the code and contribute to make it better! Happy shortening! ðŸŽ‰

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file, look at the .env.example for reference



Clone the project

```bash
  https://github.com/sharmas07/slack-url-shortener.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
